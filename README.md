# ws-cool-content-site

Demo site for nextjs workshop

Clone this repo

```
git clone https://github.com/pieterdnz/ws-cool-content-site.git

cd ws-cool-content-site

git checkout step0

yarn

yarn dev
```

step0 is the start of this course
